    About Uni-Eats

    A web application developed specifically for organization food courts.

    The main Aim of this application is to:

    • Improve service.
    • Reduce the need for queueing.
    • Improve planning both for chefs and customers.

    This application can be applicable to any organization with one or more food courts. The organizations can include but not limited to: Universities,Hospitals,Companies ,etc.
